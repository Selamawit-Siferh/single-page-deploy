# Apex Digital Systems

A modern, single-page portfolio website for Apex Digital Systems, a web development agency that builds high-performance websites and custom applications.

## Features

- **Responsive Design**: Works on desktop and mobile devices
- **Hero Section**: Rotating banner with catchy taglines
- **Services**: Custom web design, scalable web apps, SEO & optimization
- **How It Works**: Step-by-step process (Discovery → Development → Deployment)
- **Portfolio**: Showcases 3 sample projects with tech stack tags
- **Testimonials**: Customer reviews with ratings
- **About Us**: Company info, mission, vision, values, achievements
- **Contact Form**: With project type selection and Calendly integration
- **Footer**: Quick links and contact information

## Tech Stack

- **React 19** - UI library
- **Vite 8** - Build tool
- **TypeScript** - Type safety
- **CSS** - Styling

## Getting Started

### Installation

```bash
npm install
```

### Development

Start the development server:

```bash
npm run dev
```

### Build

Build for production:

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
